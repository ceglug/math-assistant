  1  """ 
  2  System configuration is one of the most important issues for making this tool more convenient for users. 
  3  By handling multiple profiles, we can edit, manage and load the profiles for our system. 
  4  By giving more choices for configuration, this makes our tool dynamic and easy to use. 
  5  Also configurations can be saved and reused on other systems by means of the profile handling mechanism. 
  6   
  7  Manage profile is used to perform profile creation and removal operations. 
  8  Edit profile is used to edit the default or predefined settings and saves these settings in the configuration files. 
  9  Load profile loads the specified configuration file, and assigns the saved settings to the current math assistant object. 
 10  All these operations are based on the configuration files in current directory. 
 11   
 12  Following user settings are added to this configuration module. 
 13   
 14      - Positioning button controls in button menu panels by user choice. 
 15      - Color Settings (Panel / Highlighting) 
 16      - Zoom level  
 17      - Highlighting Speed Settings (Menu / Menu item) 
 18      - Input settings  (Mouse left click / button click / both) 
 19      - Sound settings. 
 20      - Speech settings. 
 21      - Button Panel Settings (Horizontal / Vertical) 
 22   
 23  After editing these settings the system saves that data in a configuration file for later use. 
 24  When loading any one of this configuration file, saved settings are applied to the math assistant object.
