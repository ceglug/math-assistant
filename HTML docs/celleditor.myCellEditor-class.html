                                myCellEditor

Grid control output consist only one letter or operator or symbol inside each cell for the input. It displays the math problems worked out by the users. for that purpose Custom editor is used to have only one value for each cell. This custom class used to modify the grid control and also fulfils our requirement to perform math operations.
